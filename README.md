# Real-Time Chat Application

A full-stack, real-time messaging application built with the MERN stack and Socket.IO. It provides a responsive, WhatsApp-style interface for seamless communication between users.

## Features

- User authentication
- Real-time messaging
- Typing indicators
- Read receipts
- Last seen status
- Reply to messages
- Scheduled messages
- Conversation search
- Unread message count
- Responsive WhatsApp-style UI
- Image sharing

## Tech Stack

**Frontend:**
- React
- Tailwind CSS
- Zustand

**Backend:**
- Node.js
- Express.js
- MongoDB
- Socket.IO
- JWT

**Other:**
- Cloudinary

## Installation

```bash
git clone <repository-url>
cd <project-folder>

cd backend
npm install

cd ../frontend
npm install
```

## Run the Project

Ensure you have your MongoDB URI and necessary environment variables configured before starting the servers.

```bash
# Backend
cd backend
npm run dev

# Frontend
cd frontend
npm run dev
```

## Project Structure

- `frontend/` - React application
- `backend/` - Node.js and Express server
